# 現行WEBアプリ（スマートフォン版） 画面一覧と機能概要

このドキュメントは `now_JP_system/screen_capture_userside_mypage/現行WEBアプリSP版スクショ集/` に保存されているスクリーンショットに基づき、各画面の名称と機能の概要をまとめたものです。

## 画面一覧

| ファイルプレフィックス | 画面名 | 機能概要 |
|---|---|---|
| SP00 | LINEトーク画面 | LINEアカウントとの連携や、LINEを通じた通知・操作に関連する画面。 |
| SP01 | メンバー登録 | 新規会員登録フォーム。複数のステップに分かれており、個人情報や支払い情報などを入力する。 |
| SP02 | トップページ（未ログイン） | ログインしていないユーザー向けのトップページ。サービス紹介やログイン・新規登録への導線が配置されている。 |
| SP03 | ログイン画面 | 既存会員向けのログインページ。メールアドレスとパスワードで認証を行う。 |
| SP04 | トップページ（ログイン済） | ログイン後のトップページ。会員向けのメニューや情報が表示される。 |
| SP05 | スケジュール（スタジオ検索） | レッスンスケジュールをスタジオ（場所）で絞り込んで検索・表示する機能。 |
| SP06 | スケジュール（日付検索） | レッスンスケジュールを日付で絞り込んで検索・表示する機能。 |
| SP07 | 予約・キャンセル履歴一覧 | ユーザー自身のレッスン予約履歴およびキャンセル履歴を確認できる一覧画面。 |
| SP08 | お知らせ一覧 | 運営からのお知らせを一覧で表示する画面。 |
| SP09 | お知らせ詳細 | お知らせ一覧から選択した、個々のお知らせ内容を詳細に表示する画面。 |
| SP10 | マイページ | 会員情報、現在の予約状況、保有チケット、契約プランなどを確認・管理できる会員専用ページ。 |
