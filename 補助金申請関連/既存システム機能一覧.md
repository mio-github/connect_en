# **既存システム（国内）の機能一覧と課題**
## **Current System (Japan): Feature List and Challenges**

---

### **1. はじめに：現状のシステムが抱える根本的課題**

現在、EnDance Studioの国内事業で利用しているシステムは、日々の運営を支える一方で、事業の成長を妨げるいくつかの根本的な課題を抱えています。これらの課題は、手作業による非効率、データの分断、そして将来の拡張性の欠如に集約されます。本ドキュメントでは、具体的な機能とそれに紐づく課題を明確にし、新システム開発の必要性の根拠とします。

---

### **2. 既存システムの機能と制約**

以下の表は、現在利用しているシステムの主要機能と、それぞれが抱える制約・課題をまとめたものです。

| 機能分類 | 機能名 | 現状の運用状況と課題 |
|:---|:---|:---|
| **ダッシュボード** | ダッシュボード表示 | シンプルな情報表示のみ。経営判断に必要な分析機能やリアルタイム性に欠ける。 |
| **会員管理** | 入会管理 | 新規会員の登録は可能だが、手続きに手動の部分が多く、時間がかかる。 |
| | 会員情報管理 | 会員情報の検索・参照は可能だが、複数拠点での情報統合ができず、顧客を横断的に把握できない。 |
| | ICカード管理 | カードへの情報書き込みは可能だが、物理カードに依存しており、紛失時の再発行手続きが煩雑。 |
| **予約・レッスン** | 予約管理 | レッスンの予約は可能だが、リアルタイムでの空き状況反映が遅れることがある。 |
| | 出席管理 | レッスン毎の出席記録は取れるが、手動でのチェックイン作業が必要。 |
| | コース・月謝管理 | コース情報や月謝の検索は可能だが、複雑な料金体系やキャンペーンへの柔軟な対応が困難。 |
| **決済・POS** | POS連携 | 物販の会計は可能だが、売上データが他システムと分断されており、統合的な売上分析ができない。 |
| | 売上集計 | 日次や月次の売上集計は可能だが、単純な集計機能のみで、要因分析や未来予測は不可能。 |
| **コミュニケーション** | メール配信 | 会員への一斉メール配信は可能だが、顧客セグメントに応じたパーソナライズ配信ができず、開封率が低い。 |
| | お知らせ管理 | スタッフからのお知らせ掲載は可能だが、ターゲットを絞った情報発信ができない。 |
| **分析・レポート** | スタジオ別集計 | スタジオごとの売上は確認できるが、複数拠点を横断した比較分析や、収益性の評価が難しい。 |
| | スタジオ利用分析 | スタジオの稼働状況は確認できるが、時間帯や曜日ごとの詳細な利用パターン分析まではできない。 |

---

### **3. 総括：なぜ新システムが必要か**

既存システムは、各機能が独立して動作する「点の集まり」であり、互いに連携していません。その結果、以下のような経営上の大きな機会損失を生んでいます。

- **データドリブンな意思決定の欠如**: 経営判断が担当者の経験や勘に依存し、客観的データに基づく科学的アプローチが取れていない。
- **人件費の増大と生産性の低下**: 多くの手作業が残存し、スタッフが本来注力すべき顧客サービスや創造的な業務の時間を圧迫している。
- **顧客満足度の頭打ち**: 一人ひとりの顧客に最適化されたサービスを提供できず、顧客の定着や単価向上に繋げられていない。
- **事業拡大の足枷**: 新規出店や多角的なサービス展開に対応できる柔軟性や拡張性がなく、将来の成長機会を逃すリスクがある。

これらの課題を根本的に解決し、持続的な成長を実現するためには、分断された機能をAIによって有機的に繋ぎ、**「予測し、最適化し、パーソナライズする」新しい統合管理プラットフォーム**への刷新が不可欠です。
