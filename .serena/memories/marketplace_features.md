# ConnectEn マーケットプレイス機能

## 概要
ホットペッパービューティー型の集客・予約マーケットプレイス機能を統合し、ダンススタジオ業界の新規顧客獲得とオンライン予約を支援。

## B2C機能（消費者向け）
### スタジオ検索
- エリア検索（都道府県、市区町村、駅名）
- ジャンル検索（バレエ、ヒップホップ、ジャズ等）
- 条件絞り込み（料金、レベル、年齢、設備）
- AI推奨機能

### 予約・決済
- 体験レッスン予約
- オンライン決済（クレジットカード、PayPay、LINE Pay）
- マイページ管理
- リマインダー通知

## B2B機能（スタジオ向け）
### 出店管理
- スタジオページ編集
- SEO最適化
- 掲載プラン管理（無料/ベーシック/プレミアム）

### プロモーション
- クーポン発行
- キャンペーン設定
- メール配信
- プッシュ通知

### 分析
- アクセス解析
- 予約分析
- 売上分析
- ROI測定

## 収益モデル
### スタジオ向け
- 掲載料（月額固定）
- 成果報酬（予約成立時10-20%）
- 広告料

### 消費者向け
- 基本利用無料
- プレミアム会員（月額制）

## 実装フェーズ
- Phase 1 (3ヶ月): MVP - 基本検索・予約
- Phase 2 (6ヶ月): 決済・レビュー
- Phase 3 (9ヶ月): AI・マーケティング自動化
