## 📋 システムの構成理解

ConnectEnは**2つの異なるシステム**で構成されています。表現を間違えないよう注意してください。

---

## 🏢 1. 社内業務システム（メインシステム）

### 対象ユーザー
- **En Dance Studio の管理者・スタッフ**
- **En Dance Studio の会員（生徒）**

### 主要機能
- 会員管理
- レッスン管理
- 予約管理
- 支払い・月謝管理
- スタッフ・インストラクター管理
- 出席管理
- スクール・施設管理
- POS・小売管理
- レポート・分析

### 使用禁止表現

❌ **使ってはいけない表現**:
- 「アップグレード」「プレミアム機能」
- 「有料プラン」「無料プラン」
- 「プランを選択」「料金プラン」
- 「サブスクリプション」
- 「トライアル」「お試し」
- 「購入する」（システム機能に対して）

✅ **正しい表現**:
- 「設定」「機能」
- 「権限」「アクセス権」
- 「会員登録」「スタッフ登録」
- 「管理者設定」
- 「利用開始」

### 画面例

#### ❌ 悪い例
```
┌─────────────────────────────┐
│ アップグレード              │
│ プレミアム機能で業務効率UP! │
│ ・高度な分析機能            │
│ ・無制限のレポート          │
│ [今すぐアップグレード]      │
└─────────────────────────────┘
```

#### ✅ 良い例
```
┌─────────────────────────────┐
│ 分析機能                    │
│ 業務データを詳しく分析      │
│ ・売上分析                  │
│ ・会員動向分析              │
│ [設定を開く]                │
└─────────────────────────────┘
```

## 🚫 共通の禁止事項

### すべての画面で避けるべき表現

1. **過度な営業的表現**
   - ❌ 「今すぐ購入！」「限定特典！」
   - ✅ 「詳細を見る」「情報を確認」

2. **曖昧な表現**
   - ❌ 「詳細はこちら」「もっと見る」（何の詳細か明示する）
   - ✅ 「レッスン詳細を見る」「スタジオ情報を見る」

3. **専門用語の乱用**
   - ❌ 「KPIダッシュボード」「ROI分析」（一般ユーザー向け）
   - ✅ 「売上分析」「効果測定」

4. **英語の過度な使用**
   - ❌ 「My Page」「Dashboard」「Reservation」
   - ✅ 「マイページ」「ダッシュボード」「予約」
   - ※ 併記は可（例: マイページ / My Page）

## ✍️ コンテンツ作成チェックリスト

新しい画面やテキストを作成する際は、以下を確認してください:

### 社内業務システムの場合
- [ ] 「アップグレード」「プレミアム」などの表現を使っていないか
- [ ] 業務ツールとして適切な表現か
- [ ] スタッフ・管理者が理解しやすいか
- [ ] 会員（生徒）が混乱しない表現か

### マーケットプレイスの場合
- [ ] 一般顧客に分かりやすいか
- [ ] 体験予約の導線が明確か
- [ ] スタジオの魅力が伝わるか
- [ ] 信頼性のある表現か

### 全画面共通
- [ ] 日本語として自然か
- [ ] 簡潔で分かりやすいか
- [ ] 誤解を招く表現がないか
- [ ] アクセシビリティに配慮しているか
